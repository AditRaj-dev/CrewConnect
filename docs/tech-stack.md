# Technology Stack

Crew Connect leverages modern, performance-focused technologies designed around the central goal of delivering a frictionless, unified schedule syncing experience across Web and Mobile.

## üì° Backend Stack
- **Node.js**: Asynchronous event-driven runtime handling concurrent high-volume requests.
- **Express.js**: Fast, unopinionated web framework routing APIs and validation logic.
- **MongoDB**: Schema-flexible NoSQL data store, handling deeply nested array configurations.
- **Mongoose**: Object Data Modeling (ODM) layer abstracting complex relational aggregations.
- **Security Middlewares**: `helmet`, `express-rate-limit`, `cors` and automated JWT verification flows.

## üåê Web Client Stack
- **React 18**: Component-level declarative UI logic.
- **TypeScript**: Static typing enforcing rigorous data contracts across the codebase.
- **Tailwind CSS v4**: Utility-first CSS powering advanced glassmorphism through reactive blurs and dynamic meshes.
- **Vite**: Blazing-fast frontend tooling and hot module replacement.
- **Radix UI**: Unstyled primitive components serving as extremely accessible UI foundations.

## üì± Mobile Client Stack
- **Flutter Framework (Dart)**: True multi-platform UI toolkit compiled natively.
- **Provider**: Streamlined, predictable application state management.
- **Dio**: Robust networking HTTP client configured with retry logic and custom error interceptors for spotty connections.
- **webview_flutter**: Native interface for silently executing DOM scraping inside the app structure.
- **SharedPreferences**: Secure local storage handling offline-first application data caching.
